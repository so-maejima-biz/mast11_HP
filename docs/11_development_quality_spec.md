### 2.3 DQAccordion セクション（10工程 / 2カテゴリ）

#### 設計方針

従来の縦タイムライン（左右交互レイアウト）はスクロール量が多くUXに課題があったため、**アコーディオン（折りたたみ）方式**に変更する。さらに10工程を **2つのカテゴリ** に分類し、視覚的に区切ることで全体の構造を把握しやすくする。

#### カテゴリ構成

| カテゴリ | 英語ラベル | 日本語サブラベル | 工程 |
|----------|-----------|-----------------|------|
| 1 | DEVELOPMENT | 企画・開発 | 01〜04 |
| 2 | PRODUCTION & QUALITY | 生産・品質管理 | 05〜10 |

#### カテゴリヘッダーのスタイル

```
DEVELOPMENT ─────────────────────────
企画・開発

  [01] デザイン ...
  [02] ラスト設計 ...
  [03] ソール金型設計 ...
  [04] パターン設計 ...

                    ↕ spacing-3xl (64px)

PRODUCTION & QUALITY ────────────────
生産・品質管理

  [05] 縫製 ...
  ...
  [10] 材料物性試験 ...
```

| 要素 | 仕様 |
|------|------|
| 英語ラベル | `var(--font-size-xl)`、`font-weight: 600`、`var(--color-primary)`、`letter-spacing: 0.1em` |
| 日本語サブラベル | `var(--font-size-sm)`、`var(--color-text-secondary)` |
| 装飾線 | 英語ラベルの右側に `1px solid var(--color-border)` のラインを `flex: 1` で伸長配置 |
| カテゴリ間余白 | `var(--spacing-3xl)` (64px) |

| 項目 | 内容 |

| スクロール追従 | 展開後、アイテムが画面外にある場合は `scrollIntoView` で表示位置を調整（固定ヘッダー72px分をオフセット） |

#### 全体構造

| 項目 | 仕様 |

#### カテゴリヘッダー

| 項目 | 仕様 |
|------|------|

| 日本語サブラベル | `var(--font-size-sm)`、`var(--color-text-secondary)`、ラベル直下に配置 |
| 装飾線 | 英語ラベルの右側に flex で伸長する `1px solid var(--color-border)` のライン。親要素を `display: flex; align-items: center; gap: var(--spacing-md)` とし、ラインを `flex: 1` で配置 |

| 展開コンテンツ padding | `var(--spacing-lg)` | `var(--spacing-md)` |

#### 開閉アニメーション

| 項目 | 仕様 |
|------|------|
| 方式 | CSS `grid-template-rows: 0fr → 1fr` + `opacity` のトランジション |
| 持続時間 | 350ms |
| イージング | ease |
| chevron 回転 | 閉: 0deg → 開: 180deg、transition: transform 300ms |
| スクロール追従 | 展開後にアイテムが画面外にある場合、`scrollIntoView({ behavior: 'smooth', block: 'start' })` を実行。固定ヘッダー（72px）分を `scroll-margin-top` でオフセット |
| `prefers-reduced-motion` | アニメーション無効、即時切替 |

**実装メモ**: `grid-template-rows` 方式では、アコーディオンコンテンツを `display: grid` の子要素として配置し、`grid-template-rows` を `0fr`（閉）/ `1fr`（開）でトランジションさせる。内側の要素には `overflow: hidden` を設定する。`max-height` 方式と異なり、コンテンツの高さが未知でもスムーズにアニメーションする。

### 3.3 アニメーション

- 各アコーディオンアイテムは `data-animate` + Intersection Observer でビューポート進入時にフェードイン
- `data-animate-delay` で上から順にスタッガード表示（0, 1, 2, ... 最大5程度で打ち止め）
- 開閉アニメーションは CSS `grid-template-rows` トランジションで実装（350ms ease）
- chevron 回転は CSS transition で実装（300ms ease）
- `prefers-reduced-motion` でアニメーション無効化

### 3.4 デザイン方針（AIっぽくないデザイン）

均一すぎるグリッドを避け、以下で人間味を出す:
- DEVELOPMENT カテゴリ（01-04）と PRODUCTION & QUALITY カテゴリ（05-10）をカテゴリヘッダーで明確に区分
- 閉じた状態でもサムネイル画像を見せることで視覚的なリズムを作る
- 手書き風要素は使用しない（ブランドトーンに合わない）
