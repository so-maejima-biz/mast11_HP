## 採用文言

### ヒーロー（ファーストビュー）

| 要素 | 採用文 |
|------|--------|
| **メインコピー** | 全工程に、責任を。 |
| **サブコピー** | 企画から生産まで一貫して手がける、ものづくりのプロフェッショナル集団 |
| **CTA** | お問い合わせ / 詳しく見る |

**選定理由**：
- 「全工程に責任を持つ姿勢」を一言で表現
- 過度な装飾を避け、信頼感を重視
- 具体的な技術より「姿勢」を伝える（技術詳細は中段で）

### 中段（About）

| 要素 | 採用文 |
|------|--------|
| **見出し** | 技術で、形にする。 |
| **リード文** | 国内メーカー出身のエンジニアが、長年培った技術と経験でお客様のアイデアを具現化します。 |
| **本文1** | 私たちはスポーツシューズの企画開発生産を一貫して行ってきました。ラスト設計、ソール設計、アッパーデザインまで、すべての工程を自社で手がけることで、細部まで妥協のない品質を実現します。 |
| **本文2** | AIを活用した効率化と、少数精鋭だからこそ可能なきめ細やかな対応。企画から納品まで、ワンストップでお任せいただけます。 |

**選定理由**：
- 技術を全面に押し出しつつ、読みやすい3行程度に収める
- 「国内メーカー出身」で信頼性を示す
- 「ワンストップ」で利便性をアピール
